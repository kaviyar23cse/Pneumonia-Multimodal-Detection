# 🫁 Multimodal Explainable Deep Learning Framework for Pneumonia and COVID-19 Detection

## 📌 Project Overview

This project aims to develop an intelligent healthcare decision-support system capable of classifying chest X-ray images into:

* ✅ Normal
* ✅ Bacterial Pneumonia
* ✅ Viral Pneumonia
* ✅ COVID-19

The system utilizes an ensemble of deep learning models along with Explainable AI techniques to assist healthcare professionals in accurate diagnosis and interpretation.

---

## 🎯 Objectives

* Develop a multiclass classification framework for chest X-ray analysis.
* Differentiate between bacterial pneumonia, viral pneumonia, COVID-19, and normal cases.
* Implement ensemble deep learning techniques.
* Integrate Explainable AI (GradCAM) for model transparency.
* Develop a complete web application for prediction and visualization.
* Explore multimodal learning using image and clinical data.
